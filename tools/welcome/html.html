How to handle data in this software

Wavemaker uses a locally stored browser based database to keep your projects in. So no logins are needed :) .The plus side is you control your data, the downside is YOU control your data ...

I have a backup system that can create a ".wmdata" file you can save and even load into other devices. Your work will REMAIN in wavemaker unless you delete the data, or someone clears the browser cache!
